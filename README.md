# ComfyUI Remove Background

Remove background scenery from an image of a person. The output image is saved as an RGBA PNG. The alpha channel is included.

## Installation

- Clone this repo into custom_nodes folder.
- Install dependencies: `pip install -r requirements.txt` or if you use the portable install, run this in `ComfyUI_windows_portable/python_embeded/` folder:

  `python -m pip install -r ../ComfyUI/custom_nodes/comfyui-remove-background/requirements.txt`
